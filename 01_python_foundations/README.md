# Section 1: Python Foundations

Welcome to the first section of the Python bootcamp. It starts with **Programming 101** ideas for absolute beginners—what programming is, how execution moves step by step, and how to think in pseudocode—then moves into **Python itself**: setup, scripts, variables, types, I/O, comments, and a first **debugging** mindset.

## Learning goals

By the end of this section, you will be able to:

- Describe **what programming is** and how programs **receive, process, and output** data.
- Explain how Python code **runs** in order and why **sequence** and **state** matter.
- Write simple **pseudocode** and translate steps into Python.
- Explain **what Python is** and when it is a good tool; install it and confirm it runs.
- Run Python **interactively** and from **`.py`** script files.
- Use **variables**, core **data types**, **input/output**, and **comments**.
- Approach **errors** calmly: read messages, fix small mistakes, and retest.

3. Compare your output to the **Expected output** in the lesson.
