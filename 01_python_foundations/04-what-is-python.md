# What is Python?

## Theory

**Python** is a **high-level programming language**. “High-level” means you write instructions that look closer to human language than to the raw instructions a computer’s processor runs. A program called the **Python interpreter** reads your code and runs it.

People use Python for:

- **Scripts and automation** (rename files, process data, glue tools together)
- **Web development** (with frameworks like Django or Flask)
- **Data analysis and science** (with libraries like pandas and NumPy)
- **Machine learning** (with libraries like scikit-learn, TensorFlow, PyTorch)
- **Teaching programming** because the syntax is often clear for beginners

Python emphasizes **readability**: code is meant to be easy to read and share. That does not mean every program is simple—big systems are still complex—but the language encourages a straightforward style.

Python is **dynamically typed**: you do not declare variable types in the source the way you do in some other languages; types are attached to values at runtime (you will learn types in the next lessons).

## Beginner-friendly notes

- You are learning a **skill**, not memorizing every fact. It is normal to forget names; you can always look them up.
- **Interpreter**: runs your code line by line (in interactive mode) or from a file.
- **Syntax**: the rules of how code must be written (like grammar in a natural language).
- **Version**: Python 3 is the current line; this bootcamp assumes **Python 3.x**.

## Example

You do not need to understand every word yet. This is a tiny program:

```python
message = "Hello, Python!"
print(message)
```

## Expected output

```text
Hello, Python!
```
